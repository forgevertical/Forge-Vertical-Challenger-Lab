## 🏛️ The Mission
This repository simulates a production CI/CD environment with a critical logic vulnerability. Your objective is to achieve **Poisoned Pipeline Execution (PPE)** to exfiltrate the "Forge-Flag" from the environment secrets.

### 🕵️ Objective
1. Identify the logic flaw in the repository's GitHub Actions configuration.
2. Execute a payload that successfully captures the hidden flag.
3. Validate the integrity of your findings using the lab's evidence protocol.

---

## 🛡️ Submission Protocol
To be considered for project-based contracts with **Forge Vertical**, you must deliver a professional disclosure report to **jarrit@forgevertical.com**.

**Your report must include:**
* **Technical Summary:** A concise explanation of the vulnerability discovered.
* **Reproduction Steps:** Step-by-step instructions (CLI/Terminal based) to replicate the exploit.
* **The Flag:** The string captured from the research environment.
* **Integrity Signature:** A **SHA-256 hash** of your final exploit script/workflow.

## 📜 Safe Harbor & Rules of Engagement
* **No Destructive Actions:** Do not attempt to delete repository history or disrupt GitHub infrastructure.
* **No Lateral Movement:** Research is strictly limited to this repository's environment.
* **Ethical Disclosure:** Findings must be kept confidential and reported directly to Forge Vertical.
